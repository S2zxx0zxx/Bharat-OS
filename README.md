# BharatOS — India's AI Life Operating System 🇮🇳

India ka pehla AI assistant jo har Indian ki madad karta hai.
Legal, Govt Schemes, Health, Finance, Agriculture, Education — sab Hindi mein.

## Modules
- ⚖️ NyayBot — Legal help & rights
- 🏛️ JanSeva — Government schemes
- 🏥 Swasthya — Health information  
- 💰 Dhan — Finance & banking
- 🌾 Kisan — Agriculture
- 📚 Gyaan — Education & career

## Tech Stack
Next.js 14 · TypeScript · Tailwind CSS · Framer Motion · Gemini API · Supabase

## Setup
1. Clone repo
2. npm install
3. Copy .env.example to .env.local and fill keys
4. npm run dev

## Deploy
npx vercel --prod
